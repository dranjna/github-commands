This GitHub Cheatsheet is For me!
  1. How to upload project on github: First you have make account on github and new repository. 
                      
     * Create a local copy of a remote repository
    
                    
      # Git clone ssh://something.com/[username]/[repository_name].git:

  2. check status

    git status

  3. Adding one specific file in staging area.

    Git add [file_name.doc]

  4. Add all new and changed files to the staging area.

    git add .
  5. Commit all the required changes. 
    Git commits –m [“message for commit”]
  6. Helping for removing the file or any defined folder. 
    Git rm –r [file_name.doc]
  7. Creating a new branch with new name. 
    Git branch [branch_name]
  8. Deleting a specific branch. 
    Git branch –d [branch_name]
  9. Helping to create one specific new branch and switching entire codebase to it. 
    Git checkout -b [branch_name]
  10. Helping for cloning one remote branch and switching entire codebase to it. 
    Git checkout –b [branch_name] origin/branch_name
  11. Helps for pushing one of the created branches in your local environment to a remote directory or repository. 
    Git push origin [branch_name]
  12. Helping for pulling entire changes from the remote repository to local. 
    Git pull origin [branch_name]
